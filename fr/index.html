Questions fréquentes

Que recevrai-je exactement après l'achat ?

Vous recevez un e-mail de livraison numérique avec ce dont vous avez besoin pour utiliser la licence (en général un code et/ou des instructions pratiques, plus les références utiles à la preuve d'achat, selon le contenu réel du message envoyé).

La livraison est uniquement numérique : conservez l'e-mail et vérifiez aussi les courriers indésirables / spam si rien n'apparaît dans votre boîte de réception.

Combien de temps entre le paiement et l'e-mail avec la licence ?

En règle générale, l'e-mail arrive juste après la confirmation du paiement ; dans de rares cas, il peut falloir quelques minutes de plus pour les vérifications de paiement.

Si dans les 30 minutes vous n'avez toujours rien reçu, vérifiez le spam et écrivez à Info@amlstore.it avec le produit acheté et l'e-mail utilisé pour la commande, afin que nous puissions vérifier l'envoi.

Je n'ai jamais acheté de licence numérique : comment cela fonctionne-t-il du début à la fin ?

Choisissez le produit, finalisez la commande et le paiement avec les moyens disponibles au checkout (lorsque c'est actif, le paiement par carte est traité via Stripe).

Une fois le paiement confirmé, vous recevez la livraison par e-mail ; pour utiliser la licence, suivez l'e-mail et la fiche produit sur notre site, où vous trouverez les références correspondant à ce titre.

Le prix est plus bas qu'ailleurs : la licence est-elle conforme ? Pourquoi coûte-t-elle moins cher ?

Nous vendons des licences numériques décrites sur les fiches produit, avec des prix clairs et des conditions consultables dans les Conditions générales et la Politique de confidentialité avant l'achat.

Si vous souhaitez un retour externe, vous pouvez lire les avis publics sur Trustpilot. Pour toute question opérationnelle, écrivez à Info@amlstore.it.

Où saisir le code et comment installer le logiciel ?

L'activation et l'installation se font sur les sites et portails officiels de l'éditeur, selon les règles de ce produit.

Après l'achat, utilisez les instructions reçues par e-mail et, pour le détail pas à pas, la fiche produit sur notre site. Si quelque chose ne fonctionne pas, écrivez à Info@amlstore.it avec le produit et l'e-mail de commande.

Comment vous contacter pour un problème de commande, de paiement ou d'activation ?

Écrivez à Info@amlstore.it : c'est le canal principal indiqué aussi dans le pied de page du site.

Dans le message, indiquez le produit, l'e-mail utilisé pour la commande et une brève description du problème ; pour la confidentialité, les conditions et les retours, utilisez les liens du pied de page (Politique de confidentialité, Conditions générales, Retours et remboursements).
