FAQ

Questions fréquentes

S'agit-il d'un abonnement ou d'un achat unique ?

C'est un achat unique : la licence est perpétuelle, avec un paiement unique et aucun renouvellement.

Comment activer la licence ?

Activez le code sur setup.office.com avec votre compte Microsoft, puis installez les applications depuis office.com.

Puis-je utiliser les applications hors ligne ?

Oui : une fois les applications de bureau installées, vous pouvez travailler hors ligne ; une connexion périodique est nécessaire uniquement pour la vérification de la licence et les mises à jour.

Que se passe-t-il si je change d'ordinateur ?

La licence est liée au premier appareil sur lequel elle est activée, selon les conditions Microsoft affichées lors de l'activation.

Puis-je obtenir une facture ?

Oui, la facture électronique est disponible pour les entreprises en indiquant un numéro de TVA/SDI lors du paiement.
