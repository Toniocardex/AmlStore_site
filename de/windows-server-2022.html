Die Windows Server 2022 Standard-Lizenz deckt die auf dieser Produktseite angegebene Anzahl an Kernen ab; prüfen Sie den CAL-Bedarf je nach Anzahl der verbindenden Nutzer/Geräte.

Wie aktiviere ich die Lizenz?
