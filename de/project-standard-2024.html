FAQ

Häufig gestellte Fragen

Handelt es sich um ein Abo oder einen Einmalkauf?

Es ist ein Einmalkauf: Die Lizenz ist unbefristet, mit einmaliger Zahlung und ohne Verlängerungen.

Wie aktiviere ich die Lizenz?

Lösen Sie den Code auf setup.office.com mit Ihrem Microsoft-Konto ein und installieren Sie die Apps anschließend von office.com.

Kann ich die Apps auch offline nutzen?

Ja: Nach der Installation der Desktop-Apps können Sie offline arbeiten; eine gelegentliche Internetverbindung ist nur für die Lizenzprüfung und Updates nötig.

Was passiert, wenn ich den Computer wechsle?

Die Lizenz ist an das erste Gerät gebunden, auf dem sie aktiviert wird, gemäß den bei der Aktivierung angezeigten Microsoft-Bedingungen.

Kann ich eine Rechnung erhalten?

Ja, für Unternehmen ist an der Kasse eine elektronische Rechnung mit USt-ID/SDI-Code verfügbar.
