Preguntas frecuentes

¿La licencia es perpetua o tiene caducidad?

La licencia es perpetua (de por vida) para el dispositivo en el que se activa. No caduca ni requiere renovaciones.

¿Cómo activo la licencia de Windows 11?

Introduce la clave de producto recibida por email durante la instalación del sistema operativo o en Configuración > Activación si Windows 11 ya está instalado.

¿Puede esta licencia actualizar desde Windows 10?

Sí. Si tu PC cumple los requisitos mínimos de hardware de Microsoft, puedes usar la clave de producto para actualizar una instalación compatible de Windows 10 a Windows 11.
